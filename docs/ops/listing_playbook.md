# Margin Runner Listing Playbook

## 1. Platform Selection Logic

| Product Category | Recommended Platform | Why? |
| --- | --- | --- |
| **Electronics (Small/Standard)** | Amazon FBA | High velocity, built-in trust, Prime shipping. |
| **Home & Kitchen (New)** | Amazon FBA | Standard fulfillment, high volume. |
| **LEGO & Toys** | Amazon FBA / eBay | FBA for speed; eBay for retired/collectible sets. |
| **Collectibles / Niche / Used** | eBay | Auction format or BIN with specific buyer focus. |
| **Bulky/Heavy Items (TVs, ACs)** | Facebook Marketplace | Local pickup eliminates massive shipping costs. |
| **High Fraud Risk (Phones, etc)** | Facebook Marketplace | Face-to-face cash transaction is safest. |

## 2. Deal Qualification Tiers (Net Margin)

- **🔥 TOP DEAL (Net Margin > 40%)**: Priority acquisition. High confidence, high profit.
- **✅ GOOD DEAL (Net Margin 30-40%)**: Stable flip. Maintain consistent inventory.
- **⚠️ CAUTION (Net Margin < 30%)**: Only proceed if velocity is extremely high or effort is minimal.

## 3. Best Practices for Listing

### Amazon FBA
- **Buy Box**: Price competitively to stay in the Buy Box rotation.
- **Condition**: Only list "New" if the packaging is pristine. Use "Used - Like New" for box damage.
- **Restrictions**: Always check if you are gated for the brand before purchasing.

### eBay
- **Photos**: Use all 12 photo slots. Include close-ups of serial numbers and seals.
- **Shipping**: Use calculated shipping to avoid losing margin on distance.
- **Promotion**: Use 2-5% promoted listings for faster turnover if margins allow.

### Facebook Marketplace
- **Safety**: Meet in daylight, public places (e.g., police station parking lots).
- **Payment**: Cash or Venmo/Zelle. Avoid "protection" payments that can be reversed.
- **Clarity**: State clearly "Firm Price" if you aren't open to haggling.

## 4. Financial Buffer
Our calculations include a **10% safety buffer** on top of all costs. This is to protect against:
- Unexpected shipping price hikes.
- Small platform fee adjustments.
- Return/Damage overhead.
